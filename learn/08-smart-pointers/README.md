# 08 - Smart Pointers

智能指针：Box、Rc、RefCell、Arc、内部可变性。

## 知识点

- Box\<T\>：堆分配，递归类型
- Rc\<T\>：引用计数，单线程共享所有权
- Arc\<T\>：原子引用计数，多线程共享
- RefCell\<T\>：内部可变性（运行时借用检查）
- Deref trait：自定义解引用行为
- Drop trait：自定义清理逻辑

## 内存模型对比

| 指针 | 所有权 | 线程安全 | 可变性 | 典型场景 |
|------|--------|----------|--------|----------|
| Box\<T\> | 单一 | 是 | 可选 | 递归类型、大数据 |
| Rc\<T\> | 共享 | 否 | 不可 | 图结构、缓存 |
| Arc\<T\> | 共享 | 是 | 不可 | 多线程共享 |
| RefCell\<T\> | 单一 | 否 | 运行时 | 内部可变性 |
| Mutex\<T\> | 单一 | 是 | 运行时 | 多线程可变 |

## 代码示例

```bash
cargo run
```

输出：

```
--- Box ---
链表: 1 -> 2 -> 3 -> Nil

--- Rc ---
引用计数: 1
克隆后计数: 2
释放一个后: 1

--- RefCell ---
值: 10
修改后: 30

--- Arc + Mutex ---
线程 0: count = 1
线程 1: count = 2
线程 2: count = 3
最终 count = 3
```

# 03 - Types

Rust 类型系统：结构体、枚举、泛型、trait。

## 知识点

- 结构体（struct）与方法（impl）
- 枚举（enum）与变体携带数据
- 泛型基础
- trait 定义与实现
- derive 宏自动实现常用 trait（Debug, Clone, PartialEq）
- 常用 trait：Display, From, Into

## 代码示例

```bash
cargo run
```

# 01 - Hello Rust

入门模块：环境搭建、基本语法、Cargo 工具链。

## 知识点

- rustup / rustc / cargo 安装与使用
- `fn main()` 入口函数
- 变量绑定与 mut 可变性
- 基本类型：i32, f64, bool, char, &str, String
- `println!` 格式化输出
- `let` 绑定与 shadowing

## 代码示例

```bash
cargo run
```
